# 王健的代码库

# AAA文件夹为主要代码。  

本目录包含了基于 TradingView Pine Script v6 引擎开发的一系列高级定制化交易指标。该工具箱旨在将复杂的全球宏观经济数据、跨国流动性水位、债务周期模型与微观技术动能相结合，为长周期价值投资和中短期对冲套利提供严密的沙盘推演与决策支持。

## 📁 核心脚本目录与功能说明

### 1. `AA流动性压力指数（常驻版）`
**定位：美元体系底层水温与系统性风险雷达**
本脚本通过抓取美联储及全球主要央行最核心的资产负债表与利率数据，构建了全景式的流动性压力指数（LSI）。
* **核心监控**：深度追踪 SOFR、EFFR、RRP、TGA 以及准备金余额，通过底层管道的资金摩擦精准测算美元流动性的松紧度。
* **宏观状态识别**：智能判定“流动性黑洞”、“滞胀式危机”或“流动性泛滥”等极端宏观环境。
* **全球总量计算 (GLI)**：统合 FED、ECB、PBC、BOJ 的资产负债表规模，并经过标准化的“万亿(Trillion)”级别量纲转换，消除数据源差异带来的计算误差。

### 2. `AA对冲套利风险（常驻版）`
**定位：跨国资本流向与抛补利率平价 (CIP) 推演**
专为跨国宏观套利设计的资金流向监控器，透视华尔街无风险套利的真实成本。
* **锁汇成本计算**：自动抓取美、日、欧、中等主要经济体的短端（2年期）及长端（10年期）国债收益率，精确计算跨国交易的远期对冲成本。
* **主权风险剥离**：引入债务惩罚因子（Debt Penalty），当目标国债务率突破安全阈值时，自动扣减风险调整后的真实收益率。
* **裸敞口监控**：针对未进行锁汇的“裸多头”，结合汇率（DXY、USDJPY等）动能，监控并预警“利润反噬”与“恐慌踩踏”风险。

### 3. `AA国家债务经济数据分析（中远期）常驻版`
**定位：Dalio 债务周期与大类资产配置沙盘**
深度融合了 Ray Dalio 的“和谐去杠杆”理论与宏观基本面分析，用于评估主要经济体的长期健康度。
* **债务与增长博弈**：通过测算“名义 GDP 增长率减去 10 年期国债收益率”的利差，判定国家债务周期的扩张或去杠杆压力，并集成了基于债务率的风险平滑调节因子（debt_amp）。
* **通胀与货币动能**：监控 CPI/PPI 剪刀差以判定通胀类型（需求拉动型 vs 成本推动型），同时通过 M1/M2 剪刀差跟踪市场活钱的流向。
* **自适应 UI 面板**：内置多层级的字体大小自适应开关与灵活的 UI 布局设置，确保在不同分辨率和窗口缩放下数据的清晰呈现。

### 4. `AA RSI + MACD AA背离提示 (含 ATR 波动率)常驻版`
**定位：微观交易结构与技术面极值捕捉**
在宏观大周期的基础上，用于执行精准的买卖点择时与仓位波动率控制。
* **双重背离共振**：独立计算 RSI 和 MACD 的枢轴点（Pivot Points），精准识别常规及隐藏的顶/底背离信号。
* **ATR 波动率集成**：直观展示市场的真实波动率极值，在宏观流动性枯竭或情绪极度亢奋时，辅助判断价格动能的衰竭点。

## 🛠 技术特性与使用提示
* **底层语言**：全面采用 Pine Script v6 标准构建。
* **数据源依赖**：脚本高度依赖 TradingView 提供的 `FRED:`、`ECONOMICS:` 及跨市场债市/汇市数据源。
* **部署建议**：宏观类脚本（债务分析、流动性指数）建议在 `日线(D)` 或更高时间级别下加载使用，以过滤日内噪音，捕捉真正的宏观趋势。

# BBB为分支代码
主要内容都和AAA差不多，但BBB更细分。如果AAA代码需要对某一个指标进行精对，可以用BBB指标。  

本目录是 AAA 核心代码库的**分支与深度扩展版**。

**【定位说明】**
AAA 文件夹中的“常驻版”指标旨在提供全天候、跨维度的宏观与技术面综合监控；而 **BBB 文件夹则侧重于“特定时间周期的极致细分”**。当 AAA 指标发出预警或信号时，如果你需要对某一个具体的宏观逻辑（如中期货币周期、远期国家债务）进行深度“精对”和剖析，请调用本目录下的特定指标。

## 📁 细分脚本目录与功能说明

### 1. `宏观全景 V7.9 (中期细分)`
**定位：中期（3-12个月）货币周期与商业周期精调器**
将宏观视角聚焦于中期的流动性活化程度与衰退预警，专为捕捉跨季度的趋势拐点设计。
* **货币与利差共振**：深度解析 M1/M2 剪刀差动能（EMA6平滑）与 10Y-2Y 期限利差的陡峭/倒挂状态，判定资金是从实体流向金融，还是正在被“死水”困住。
* **通胀与滞胀剥离**：细分“需求拉动型”、“成本推动型”与“通缩压力型”通胀，并独创滞胀压力评分（Stagflation Score），精准识别企业盈利（PPI）与实际利率的双杀区间。
* **高密度精简看板**：采用紧凑的 6×8 矩阵 UI 布局，在有限的图表空间内一口气呈现货币环境、流动性压力、铜金比风险偏好以及系统总评，适合实盘中快速盯盘。

### 2. `达利欧宏观看板 V26 (远期细分)`
**定位：远期（1-10年）国家债务周期与主权健康度沙盘**
将 Ray Dalio 的《债务危机》理论进行极致的量化与可视化，用于推演大国博弈下的底层资产根基是否稳固。
* **四大经济体横向对比**：直接在图表上并行绘制中美日欧（US, CN, JP, EU）四国的债务/GDP走势与利息压力曲线，全球主权债务风险一览无余。
* **双模态生长推演**：
    * **模式A (增长驱动)**：精准测算“名义 GDP 增长率”与“10年期国债收益率”的利差，透视国家是否具备“和谐去杠杆”的增长潜力。
    * **模式B (债务反噬)**：计算总债务率叠加真实融资成本后的“总计利息压力”，当该数值突破红色极值时，预警必然发生的流动性危机或恶性印钞。
* **五级动态诊断**：从“强扩张”到“危机”的 5 级严格评分系统，自动剔除短期数据噪音，直击国家债务周期的真实阶段。

## ⚙️ 联动使用指南 (AAA ↔ BBB)
1. **日常挂机**：在 TradingView 主图表中挂载 `AAA` 的常驻版指标，保持宏观预警雷达和技术背离提示的常开状态。
2. **中期诊断**：当 AAA 提示“宏观状态转换”（例如从复苏转入过热或衰退），打开 `BBB 宏观全景 (中期细分)`，通过 M1 动能和铜金比确认活钱的具体流向。
3. **远期排雷**：在做长线价值投资（如买入某国长期国债或宽基指数）前，务必打开 `BBB 达利欧宏观看板 (远期细分)`，确认目标国是否正处于“负利差”的债务死局中，避免陷入主权级别的杀估值陷阱。
